    datar.dplyr.funcs.across(_data, _cols=None, _fns=None, _names=None, **kwargs)

    </>

    Apply the same transformation to multiple columns

    Parameters
    • _data (DataFrame or DataFrameGroupBy) — The dataframe
    • _cols (iterable of str, optional) — The columns
    • _fns (Union(callable, iterable of callable, , nonetype), optional) — Functions to apply to each of the selected columns.
    • _names (str, optional) — A glue specification that describes how to name the output columns. This can use {_col} to stand for the selected column name, and {_fn} to stand for the name of the function being applied. The default (None) is equivalent to {_col} for the single function case and {_col}_{_fn} for the case where a list is used for _fns. In such a case, {_fn} is 1-based. To use 1-based index, use {_fn0}
    • *args, **kwargs — Arguments for the functions
    Returns (Across)

    A dataframe with one column for each column in _cols and each function in _fns.

    datar.dplyr.funcs.recode(series, *args, _default=None, _missing=nan, **kwargs)

    </>

    Recode a vector, replacing elements in it

    Parameters
    • series (iterable of any) — A vector to modify
    • _default (any, optional) — If supplied, all values not otherwise matched will be given this value. If not supplied and if the replacements are the same type as the original values in series, unmatched values are not changed. If not supplied and if the replacements are not compatible, unmatched values are replaced with NA.
    • _missing (any, optional) — If supplied, any missing values in .x will be replaced by this value.
    • *args, **kwargs — replacements
    Returns (iterable of any)

    The vector with values replaced
